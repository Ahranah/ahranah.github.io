## 자동 타입 변환

허용 범위가 작은 변수에 큰 값을 대입할 때

byte < short, char < int < long < float < double

- int에 실수 넣으면 자동 타입 변환
- char 타입은 유니코드값(정수) 대입
- byte, short은 피연산자로 사용되면 int로 자동 변환되어 연산
- double이 있는 경우 해당 식 자료형 모두 double로 자동 변환
- String: 정수 + 문자열 = 문자열 / 정수 + 정수 + 문자열 = 정수 합계 후 문자열로 결합

 

## 캐스팅 연산자 ()

허용 범위가 큰 변수를 허용 범위가 작은 변수형으로 쪼개서 저장: 작은 변수형으로 표현이 가능할 때만 가능

```java
int intValue = 65;
char charValue = (char)intValue;
System.out.println(charValue); 

// 'A'
```

**printf 문법:** %와 conversation 필수 작성

```java
System.out.printf("이름: %1$s, 나이: %2$d", "김자반", 30);

System.out.printf("이름: %s", "김자반");
System.out.printf("나이: %d", 30);
```
